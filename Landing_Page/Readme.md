Task: Build a Landing Page for EduLearn
EduLearn is an online learning platform (like Coursera). Use only HTML + CSS.
Required Sections:

Navbar — Logo + 3 navigation links + "Get Started" button
Hero — Large heading, subtext, one CTA button

Only HTML + CSS (no framework, no library)
Must be responsive (should look good on mobile too)
Colors and fonts are your choice — go with what looks good

Time limit: 90 minutes
